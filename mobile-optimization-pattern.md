# Mobile Optimization Pattern for Industry Pages

## Key Changes Applied:

### 1. Container Padding
- `px-6` → `px-4 md:px-6`

### 2. Buttons
- `px-8 py-4` → `w-full sm:w-auto px-6 sm:px-7 py-3 sm:py-3.5`
- Add `hover:shadow-lg` for primary
- Add `hover:bg-gray-50` for secondary

### 3. Button Containers
- `flex flex-col sm:flex-row gap-4` → `flex flex-col sm:flex-row gap-3 md:gap-4`

### 4. Section Headings (H2)
- `text-3xl md:text-4xl` → `text-2xl md:text-4xl`

### 5. Body Text
- `text-lg` → `text-sm md:text-lg`

### 6. Feature Titles (H3)
- `text-2xl` → `text-xl md:text-2xl`

### 7. Grids
- `grid md:grid-cols-3 gap-6` → `grid grid-cols-1 md:grid-cols-3 gap-4 md:gap-6`
- `grid md:grid-cols-2 gap-12` → `grid grid-cols-1 md:grid-cols-2 gap-8 md:gap-12`

### 8. Card Padding
- `p-6` → `p-5 md:p-6`
- `p-8` → `p-6 md:p-8`

### 9. Icons
- `w-8 h-8` → `w-7 h-7 md:w-8 md:h-8`
- `w-5 h-5` → `w-4 h-4 md:w-5 md:h-5`

### 10. Spacing
- `mb-12` → `mb-8 md:mb-12`
- `space-y-16` → `space-y-12 md:space-y-16`
- `space-y-4` → `space-y-3 md:space-y-4`

### 11. List Item Text
- Add `text-sm md:text-base` to list items

### 12. CTA Section Headings
- `text-3xl md:text-5xl` → `text-2xl md:text-5xl`

### 13. CTA Section Text
- `text-lg` → `text-sm md:text-lg`

### 14. Stat Numbers
- `text-5xl` → `text-4xl md:text-5xl`
